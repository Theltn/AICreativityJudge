# AI Creativity Judge

A deep learning framework that evaluates the creative writing capabilities of Large Language Models (LLMs). Trained exclusively on human-written fiction from the Reddit WritingPrompts dataset (~272k stories), scored by a deterministic 5-dimension creativity rubric, and used to compare stories generated by ChatGPT, Claude, Gemini, Perplexity, and Copilot.

## Installation (Windows + NVIDIA GPU)

### 1. Clone the Repository

```bash
git clone https://github.com/Theltn/AICreativityJudge.git
cd AICreativityJudge
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate
```

> You should see `(venv)` at the beginning of your terminal prompt.

### 3. Install PyTorch with CUDA Support

Go to https://pytorch.org/get-started/locally/ and generate the install command for your setup:
- OS: **Windows**
- Package: **pip**
- Compute Platform: **CUDA 12.4** (or whatever matches your driver)

It will give you something like:

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
```

**Run that command first** before the next step.

### 4. Verify CUDA is Working

```bash
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}'); print(f'GPU: {torch.cuda.get_device_name(0)}') if torch.cuda.is_available() else print('NO GPU DETECTED')"
```

You should see:
```
CUDA available: True
GPU: NVIDIA GeForce RTX 5070
```

> ⚠️ If it says `CUDA available: False`, your CUDA Toolkit or GPU drivers are not installed correctly. Fix this before continuing.

### 5. Install All Other Dependencies

```bash
pip install -r requirements.txt
```

This installs:
- `pandas`, `numpy`, `pyarrow`, `fastparquet` — data handling
- `spacy`, `nltk`, `vaderSentiment` — NLP / text scoring
- `transformers`, `accelerate` — Hugging Face (RoBERTa model)
- `scikit-learn` — evaluation metrics
- `matplotlib`, `seaborn` — visualization
- `datasets`, `huggingface_hub` — dataset downloading
- `jupyter` — notebook support

### 6. Download the spaCy Language Model

```bash
python -m spacy download en_core_web_sm
```

### 7. Download NLTK Data

```bash
python -c "import nltk; nltk.download('punkt'); nltk.download('punkt_tab')"
```

### 8. Create Required Directories

```bash
mkdir data\raw
mkdir data\processed
mkdir data\models
```

### 9. Verify Installation

Run this to confirm everything is working:

```bash
python -c "
import pandas, numpy, torch, spacy, nltk, transformers, sklearn
from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
print('All imports successful!')
print(f'PyTorch: {torch.__version__}')
print(f'Transformers: {transformers.__version__}')
print(f'CUDA: {torch.cuda.is_available()}')
nlp = spacy.load('en_core_web_sm')
print(f'spaCy model loaded')
print('Setup complete!')
"
```

## Pipeline Overview

```
[1] Download Data → [2] Preprocess → [3] Score 272k Stories → [4] Split → [5] Train Models
```

---

## Step 1: Download Dataset

Downloads the Reddit WritingPrompts dataset from Hugging Face (~272k train + validation + test splits):

```bash
python scripts/data_collection.py
```

**Output:** `data/raw/writing_prompts_*.parquet`

---

## Step 2: Preprocess Data

Cleans text (removes URLs, markdown, Reddit tags) and filters stories under 100 words:

```bash
python src/data_preprocessing.py
```

**Output:** `data/processed/writing_prompts_full_cleaned.parquet`

---

## Step 3: Score Dataset

Runs all 272k stories through the 5-dimension creativity rubric. This takes **~2.5 hours**:

```bash
python scripts/score_dataset.py
```

The script has **checkpointing** — if interrupted, re-run the same command and it resumes where it left off.

**Output:** `data/processed/writing_prompts_scored.parquet`

### The 5 Scoring Dimensions

| Dimension | Weight | What it measures |
|---|---|---|
| Lexical Richness | 20% | Vocabulary diversity (TTR + hapax legomena) |
| Syntactic Complexity | 15% | Sentence sophistication (parse depth) |
| Novelty | 25% | TF-IDF surprise vs corpus average |
| Imagery | 20% | Sensory word density (~200 word lexicon) |
| Narrative Dynamics | 20% | Sentiment arc variance (VADER) |

---

## Step 4: Sanity Check & Create Splits

Validates that the scores make sense and creates train/val/test splits:

```bash
python scripts/sanity_check_and_split.py
```

**Output:**
- `data/processed/train.parquet` (190,805 stories — 70%)
- `data/processed/val.parquet` (40,887 stories — 15%)
- `data/processed/test.parquet` (40,887 stories — 15%)

---

## Step 5: Train Models

> ⚠️ **You must have the 3 split files** (`train.parquet`, `val.parquet`, `test.parquet`) in `data/processed/` before running training. If you're on a different machine than where you scored the data, transfer these files manually.

### Option A: Local Training (Recommended for NVIDIA GPU)

Trains both MLP baseline and RoBERTa in one script:

```bash
python scripts/train_local.py
```

| Model | Time (RTX 5070) | Time (Colab T4) |
|---|---|---|
| MLP Baseline | ~1 minute | ~1 minute |
| RoBERTa | ~45 min – 1.5 hrs | ~2-4 hours |

**Output:** Saved to `data/models/`:
- `mlp_best.pt` — MLP weights
- `roberta_creativity_model/` — Full RoBERTa model
- `*_training_curves.png` — Training loss plots
- `*_scatter.png` — Predicted vs true scatter plots
- `*_metrics.json` — MSE, MAE, R² scores

### Option B: Google Colab

If you don't have a local GPU:

1. Upload `train.parquet`, `val.parquet`, `test.parquet` to Google Drive under `AICreativityJudge/data/`
2. Upload `notebooks/02_MLP_Baseline.ipynb` to Colab → Run All
3. Upload `notebooks/03_RoBERTa_Finetuning.ipynb` to Colab → Set runtime to **T4 GPU** → Run All

See `Docs/Colab_Training_Instructions.md` for detailed steps.
